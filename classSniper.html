MU Class Sniper

Perl script to alert of class openings at Monmouth University.

Perl
Mechanize

One thing that all college students have in common is procrastination, and that flaw makes registering for the upcoming semester a pain. Once you are able to find time to register for the courses you want they are already full, this script will fix that issue. MU Class Sniper is designed to monitor a specific course and section, then alert you by email when the class has opened.
